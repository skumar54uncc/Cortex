Cortex

Cortex keeps indexed content on your device. Indexing makes no network calls—the embedding model ships bundled under dist/models/ (run npm run prepare-model before building). Optional Cloud Chat may send retrieved snippets to Google Gemini only when you enable it and add an API key.

Import from browser history

Backfill pages you already visited. Cortex requests each URL over the network (static HTML only — many single-page apps won’t fully render). Your blocklist, allowlist, and sensitive URL rules still apply. Runs locally except for those fetches; embeddings stay on-device. Multiple URLs are fetched in parallel so scans finish much faster; indexing and embeddings still run locally afterward.
